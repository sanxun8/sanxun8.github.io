## 一元二次方程的定义与性质:

- 定义:ax^2 + bx + c = 0 (a ≠ 0)
- 一元二次方程的根的性质

## 一元二次方程的解法:

- 配方法
- 公式法
- 因式分解法

## 一元二次方程的判别式:

- 判别式的定义
- 判别式与方程根的关系

## 一元二次方程的应用:

- 应用背景:几何、物理、经济等
- 建立数学模型并求解

## 实例练习:

- 求解方程：2x^2 - 3x - 5 = 0
- 一个矩形的长和宽的差是3米,面积是24平方米,求长和宽。
