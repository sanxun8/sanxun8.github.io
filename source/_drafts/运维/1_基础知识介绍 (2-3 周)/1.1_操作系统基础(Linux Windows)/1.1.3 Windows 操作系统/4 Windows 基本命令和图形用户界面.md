---
title: Windows 基本命令和图形用户界面
---

Windows 操作系统提供了丰富的命令行界面和图形用户界面,用户可以根据需求选择使用:

## 命令行界面 (Command Prompt):

Windows 命令提示符(cmd.exe)是一个基于文本的界面,用户可以通过键盘输入命令进行各种操作,主要包括:

### 文件和目录管理命令:

- dir: 列出当前目录的文件和子目录
- cd: 切换到指定目录
- copy: 复制文件
- del: 删除文件

### 系统管理命令:

- ipconfig: 查看网络配置信息
- ping: 测试网络连接
- shutdown: 关机、重启电脑

### 其他实用命令:

- cls: 清屏
- help: 查看命令帮助
- exit: 退出命令提示符

命令行界面适用于需要批量操作、自动化脚本或对系统底层进行管理的场景。

## 图形用户界面 (Graphical User Interface, GUI):

Windows 提供了直观的图形用户界面,通过鼠标点击和拖拽等操作进行文件管理和系统设置,主要包括:


### 开始菜单:

- 包含应用程序、文档、设置等快捷入口
- 用户可以在此搜索、启动程序

### 任务栏:

- 显示当前正在运行的应用程序
- 用户可以在此快速切换和管理窗口

### 文件资源管理器:

- 提供直观的文件和目录浏览界面
- 支持文件/文件夹的创建、复制、删除等操作

### 控制面板:

- 集中管理系统设置和首选项
- 用户可以在此配置硬件、网络、安全等功能

图形用户界面更加直观友好,适用于日常办公、娱乐等场景,是大多数Windows用户的首选操作方式。

综合使用命令行和图形界面,可以充分发挥 Windows 操作系统的功能和灵活性,提高工作效率。
