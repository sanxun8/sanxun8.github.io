## 如何打开事件查看器

- 使用开始菜单
  1. 点击 Windows 开始菜单。
  2. 在搜索栏中输入“事件查看器”或“Event Viewer”。
  3. 选择“事件查看器”应用程序。

- 使用运行对话框
  1. 按下 Win + R 键打开运行对话框。
  2. 输入 eventvwr 并按下 Enter 键。

- 使用控制面板
  1. 打开控制面板。
  2. 转到“系统和安全”。
  3. 点击“管理工具”。
  4. 选择“事件查看器”。

## 事件查看器界面

事件查看器的界面主要包括三个部分：

- 左侧窗格（导航窗格）：包含事件日志的树状结构，包括“自定义视图”、“Windows 日志”（应用程序、安全、设置、系统和转发的事件）、“应用程序和服务日志”。
- 中间窗格（详细信息窗格）：显示选定日志的事件列表。
- 右侧窗格（操作窗格）：提供各种操作选项，例如创建自定义视图、筛选当前日志、导出日志等。

## 查看事件日志

1. 选择日志类型

- 在左侧窗格中展开“Windows 日志”。
- - 选择要查看的日志类型（应用程序、安全、设置、系统或转发的事件）。

2. 查看事件详情

- 在中间窗格中，双击要查看的事件。
- 事件详情窗口会显示事件的源、级别、日期和时间、事件 ID、任务类别、用户、计算机和详细信息。

## 筛选和查找事件

1. 筛选当前日志

- 在右侧窗格中选择“筛选当前日志”。
- 在筛选窗口中，设置筛选条件（例如日期范围、事件级别、事件源、事件 ID 等）。
- 点击“确定”应用筛选条件。

2. 查找事件

- 在右侧窗格中选择“查找”。
- 输入要查找的关键词或事件 ID。
- 点击“查找下一个”进行搜索。

## 创建自定义视图

1. 创建新视图

- 在左侧窗格中右键点击“自定义视图”。
- 选择“创建自定义视图”。

2. 设置自定义视图

- 在自定义视图窗口中，设置时间范围、事件级别、日志类型、事件源等条件。
- 点击“确定”。

3. 保存自定义视图

- 为自定义视图命名，并选择保存位置。
- 点击“确定”。

##  导出和导入事件日志

1. 导出事件日志

- 在中间窗格中选择要导出的事件。
- 在右侧窗格中选择“保存已筛选的日志文件”。
- 选择保存位置和文件名，点击“保存”。

2. 导入事件日志

- 在左侧窗格中右键点击“自定义视图”。
- 选择“导入自定义视图”。
- 选择要导入的日志文件并点击“打开”。

## 常见事件日志类型

- 应用程序日志：包含应用程序事件（如软件崩溃、错误信息）。
- 安全日志：包含安全相关事件（如登录尝试、权限变更）。
- 系统日志：包含系统级事件（如驱动程序故障、硬件问题）。
- 设置日志：包含设置更改事件。
- 转发的事件：包含从其他计算机转发的事件。

## 使用事件查看器进行故障排除

- 查找错误事件：查找标记为“错误”或“警告”的事件，以识别潜在问题。
- 查看详细信息：双击事件以查看详细信息，帮助理解事件的原因和影响。
- 搜索解决方案：根据事件 ID 和详细信息，在网上搜索解决方案或参考 Microsoft 的官方文档。

通过使用事件查看器，可以有效地监控和管理系统事件，帮助识别和解决计算机问题，确保系统的正常运行。
