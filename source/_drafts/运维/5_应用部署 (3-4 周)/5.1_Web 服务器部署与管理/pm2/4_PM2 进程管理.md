## 安装和初始化:

使用 npm 全局安装 PM2:

```shell
npm install -g pm2
```

初始化 PM2 并设置开机自启动:

```shell
pm2 startup
```

## 基本用法:

启动应用程序:

```shell
pm2 start app.js
```

列出所有运行中的进程:

```shell
pm2 list
```

监控进程状态:

```shell
pm2 monit
```

停止进程:

```shell
pm2 stop app
```

重启进程:

```shell
pm2 restart app
```

删除进程:

```shell
pm2 delete app
```

查看进程日志:

```shell
pm2 logs app
```

## 配置文件:

使用 pm2 init 命令生成配置文件模板:

```shell
pm2 init
```

在配置文件中设置应用程序的启动脚本、环境变量等。

使用 pm2 start ecosystem.config.js 命令启动应用程序。

## 负载均衡:

PM2 支持内置的负载均衡功能:

```shell
pm2 start app.js -i max  # 启动最大数量的进程
```

也可以配合 Nginx 等反向代理实现负载均衡。

## 部署脚本:

PM2 可以自动化部署过程:

```shell
pm2 deploy ecosystem.config.js production setup
pm2 deploy ecosystem.config.js production
```

在配置文件中设置部署目标主机、命令等。

## 监控和报警:

PM2 提供了 API 和 web 界面进行监控:

```shell
pm2 monit
```

可以配合第三方工具如 Keymetrics 进行监控报警。

以上就是 PM2 的一些基本用法和功能介绍。PM2 是一个非常强大的进程管理工具,广泛应用于 Node.js 生产环境中。如果您有任何其他问题,欢迎继续询问。
