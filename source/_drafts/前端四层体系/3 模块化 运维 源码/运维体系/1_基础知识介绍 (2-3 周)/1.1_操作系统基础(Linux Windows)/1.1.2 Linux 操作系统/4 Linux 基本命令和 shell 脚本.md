---
title: Linux 基本命令和 shell 脚本
---

Linux 的基本命令和 shell 脚本是 Linux 用户必须掌握的基础知识,主要包括以下内容:


## 基本命令:

- ls: 列出目录内容
- cd: 切换工作目录
- mkdir: 创建目录
- rm: 删除文件或目录
- cp: 复制文件或目录
- mv: 移动或重命名文件或目录
- cat: 查看文件内容
- grep: 在文件中搜索字符串
- chmod: 修改文件或目录权限
- sudo: 以 root 用户权限执行命令

## 进程管理命令:

- ps: 查看系统进程
- kill: 终止进程
- top: 实时监控系统进程

## 网络管理命令:

- ping: 测试网络连通性
- ifconfig: 查看和配置网络接口
- wget: 下载文件
- ssh: 远程登录到其他主机

## Shell 脚本编写:

- Bash 是 Linux 默认的 shell 程序
- 编写 shell 脚本包括变量定义、条件判断、循环等
- 常见的 shell 脚本用途包括系统管理、文件处理、自动化任务等

## Shell 脚本基本语法:

- 脚本以 #!/bin/bash 开头
- 变量定义使用 var=value 格式
- 条件判断使用 if-then-else 语句
- 循环使用 for 或 while 语句
- 函数定义使用 function name() { ... } 格式

以上是 Linux 基本命令和 shell 脚本的主要内容。掌握这些基础知识对于熟练使用 Linux 系统和进行自动化运维非常重要。
