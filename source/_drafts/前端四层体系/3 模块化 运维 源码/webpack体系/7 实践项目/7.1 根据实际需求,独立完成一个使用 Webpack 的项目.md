---

很好的要求,下面我将根据实际需求,独立完成一个使用 Webpack 的项目示例。

假设我们需要开发一个简单的 Web 应用程序,包含以下需求:

- 主要功能是一个计算器,可以进行加减乘除运算。
- 应用程序使用 React 作为前端框架。
- 需要支持 ES6/ES7 语法。
- 需要支持 SCSS 作为 CSS 预处理器。
- 需要实现模块热替换(HMR)功能,提高开发效率。
- 需要支持生产环境的代码优化和压缩。

下面我们来一步步实现这个项目:

## 初始化项目

创建项目目录,并初始化 npm 项目:

```shell
mkdir react-calculator
cd react-calculator
npm init -y
```

安装 React 和 React DOM:

```shell
npm install react react-dom
```

## 安装和配置 Webpack

安装 Webpack 及其相关依赖:

```shell
npm install --save-dev webpack webpack-cli webpack-dev-server html-webpack-plugin css-loader style-loader sass-loader node-sass babel-loader @babel/core @babel/preset-env @babel/preset-react
```

创建 Webpack 配置文件 webpack.config.js:

```javascript
const path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');

module.exports = {
  entry: './src/index.js',
  output: {
    path: path.resolve(__dirname, 'dist'),
    filename: 'bundle.js'
  },
  module: {
    rules: [
      {
        test: /\.js$/,
        exclude: /node_modules/,
        loader: 'babel-loader',
        options: {
          presets: ['@babel/preset-env', '@babel/preset-react']
        }
      },
      {
        test: /\.scss$/,
        use: ['style-loader', 'css-loader', 'sass-loader']
      }
    ]
  },
  plugins: [
    new HtmlWebpackPlugin({
      template: './src/index.html'
    })
  ],
  devServer: {
    contentBase: './dist',
    hot: true
  }
};
```

## 创建应用程序源代码

在 src 目录下创建 index.js 和 App.js 文件:

```javascript
// src/index.js
import React from 'react';
import ReactDOM from 'react-dom';
import App from './App';

ReactDOM.render(<App />, document.getElementById('root'));

// src/App.js
import React, { useState } from 'react';
import './App.scss';

const App = () => {
  const [result, setResult] = useState(0);
  const [num1, setNum1] = useState(0);
  const [num2, setNum2] = useState(0);

  const handleCalculate = (operator) => {
    switch (operator) {
      case '+':
        setResult(num1 + num2);
        break;
      case '-':
        setResult(num1 - num2);
        break;
      case '*':
        setResult(num1 * num2);
        break;
      case '/':
        setResult(num1 / num2);
        break;
      default:
        setResult(0);
    }
  };

  return (
    <div className="app">
      <h1>React Calculator</h1>
      <div className="calculator">
        <input
          type="number"
          value={num1}
          onChange={(e) => setNum1(parseFloat(e.target.value))}
        />
        <input
          type="number"
          value={num2}
          onChange={(e) => setNum2(parseFloat(e.target.value))}
        />
        <div className="buttons">
          <button onClick={() => handleCalculate('+')}>+</button>
          <button onClick={() => handleCalculate('-')}>-</button>
          <button onClick={() => handleCalculate('*')}>*</button>
          <button onClick={() => handleCalculate('/')}>/</button>
        </div>
        <div className="result">Result: {result}</div>
      </div>
    </div>
  );
};

export default App;
```

在 src 目录下创建 App.scss 文件:

```scss
.app {
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;

  .calculator {
    display: flex;
    flex-direction: column;
    align-items: center;
    padding: 20px;
    border: 1px solid #ccc;
    border-radius: 5px;

    input {
      margin: 10px;
      padding: 5px;
      font-size: 16px;
    }

    .buttons {
      margin: 10px;

      button {
        margin: 0 5px;
        padding: 5px 10px;
        font-size: 16px;
      }
    }

    .result {
      margin-top: 10px;
      font-size: 18px;
      font-weight: bold;
    }
  }
}
```

在 src 目录下创建 index.html 文件:

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>React Calculator</title>
</head>
<body>
  <div id="root"></div>
</body>
</html>
```

## 运行开发服务器和构建

在 package.json 中添加 npm 脚本:

```json
{
  "scripts": {
    "start": "webpack-dev-server --mode development",
    "build": "webpack --mode production"
  }
}
```

运行开发服务器:

```shell
npm start
```


构建生产版本:

```shell
npm run build
```
