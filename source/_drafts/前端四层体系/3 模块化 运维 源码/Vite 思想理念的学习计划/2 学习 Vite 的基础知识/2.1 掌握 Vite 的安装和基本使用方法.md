好的,让我们来学习 Vite 的安装和基本使用方法。


## 安装 Vite

- 您可以使用 npm 或 yarn 安装 Vite:

```bash
复制
# 使用 npm
npm install -g vite

# 使用 yarn
yarn global add vite
安装完成后,您就可以在命令行中使用 vite 命令了。
```

## 创建一个新的 Vite 项目

- 运行以下命令创建一个新的 Vite 项目:

```bash
复制
# 使用 npm
npm init vite@latest my-project --template vanilla

# 使用 yarn
yarn create vite my-project --template vanilla
```

- 这个命令会创建一个名为 my-project 的新文件夹,并使用 "vanilla" 模板初始化项目。
- 您也可以选择其他模板,如 React、Vue 或 Svelte 等。

## 启动开发服务器

- 进入刚刚创建的项目目录:

```bash
cd my-project
```

- 安装依赖:

```bash
# 使用 npm
npm install

# 使用 yarn
yarn
```

- 启动开发服务器:

```bash
# 使用 npm
npm run dev

# 使用 yarn
yarn dev
Vite 会启动一个开发服务器,并在终端中打印出访问 URL。
```

## 构建项目

- 当您准备部署项目时,可以使用以下命令进行构建:

```bash
# 使用 npm
npm run build

# 使用 yarn
yarn build
```

- Vite 会将您的项目打包成production-ready的静态资源,位于 dist/ 目录下。

这就是 Vite 的基本安装和使用方法。通过这些步骤,您可以快速创建一个新的 Vite 项目,并启动开发服务器进行开发。Vite 的其他高级功能,如插件系统、配置文件等,我们将在后续的学习中逐一介绍。请根据实际需求继续探索 Vite 的更多使用技巧。
