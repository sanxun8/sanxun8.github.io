---
title: 14.1.5 Comment 类型
---

DOM 中的注释通过 Comment 类型表示。 Comment 类型的节点具有以下征：

- nodeType 等于 8；
- nodeName 值为 "#comment"；
- nodeValue 值为注释的内容；
- parentNode 值为 Document 或 Element 对象；
- 不支持子节点。

comment 类型与 Text 类型继承同一个基类（CharacterData），因此拥除 splitText() 之外 Text 节 点所有的字符串操作方法。与 Text 类型相似，注释的实际内容以通过 nodeValue 或 data 属性获得。

注释节点可以作为父节点的子节点来访问。比如下面的 HTML 代码：

```html
<div id="myDiv"><! [CDATA[This is some content.]]></div>
```
