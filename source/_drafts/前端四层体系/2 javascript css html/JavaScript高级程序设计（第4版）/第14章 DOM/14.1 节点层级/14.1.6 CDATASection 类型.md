---
title: 14.1.6 CDATASection 类型
---

CDATASection 类型表示 XML 中特有的 CDATA 区块。 CDATASection 类型承 Text 类型，因此拥有包括 splitText() 在内的所有字符串操作方法。 CDATASection 类型的节点具有以下特征：

- nodeType 等于 4；
- nodeName 值为 "#cdata-section"；
- nodeValue 值为 CDATA 区块的内容；
- parentNode 值为 Document 或 Element 对象；
- 不支持子节点。
