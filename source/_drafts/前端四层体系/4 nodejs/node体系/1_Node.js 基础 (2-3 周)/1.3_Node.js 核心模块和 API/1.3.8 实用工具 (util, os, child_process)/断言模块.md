---
title: 断言模块
---

assert 模块是 Node.js 中用于编写测试和进行断言的模块。它提供了一组简单的断言函数来测试代码是否按预期运行。

## 常用的断言方法

以下是一些常用的 assert 方法：

- assert.strictEqual(actual, expected[, message])

测试 actual 是否严格等于 expected。如果不相等，则抛出一个错误。

```javascript
assert.strictEqual(1, 1); // 不会抛出错误
assert.strictEqual(1, 2); // 抛出错误：1 !== 2
```

- assert.deepStrictEqual(actual, expected[, message])

测试 actual 和 expected 是否深度严格相等。适用于比较对象和数组。

```javascript
assert.deepStrictEqual({ a: 1 }, { a: 1 }); // 不会抛出错误
assert.deepStrictEqual({ a: 1 }, { a: 2 }); // 抛出错误：{a: 1} !== {a: 2}
```

- assert.notStrictEqual(actual, expected[, message])

测试 actual 是否不严格等于 expected。如果相等，则抛出一个错误。

```javascript
assert.notStrictEqual(1, 2); // 不会抛出错误
assert.notStrictEqual(1, 1); // 抛出错误：1 === 1
```

- assert.throws(block[, error][, message])

测试函数 block 是否抛出一个错误。error 可以是构造函数、正则表达式或自定义验证函数。

```javascript
assert.throws(
  () => {
    throw new Error("错误消息");
  },
  Error // 可以是具体的错误类型或验证函数
);
```

- assert.doesNotThrow(block[, message])

测试函数 block 是否不抛出错误。如果抛出错误，则抛出断言错误。

```javascript
assert.doesNotThrow(() => {
  // 没有错误抛出
});

assert.doesNotThrow(() => {
  throw new Error("错误消息"); // 抛出断言错误
});
```

- assert.ok(value[, message])

测试是否 value 为真

```javascript
assert.ok(true); // 没有错误抛出

assert.ok(); // 抛出断言错误
```

## 示例

下面是一个完整的例子，展示如何使用 assert 模块进行测试：

```javascript
const assert = require("assert");

function add(a, b) {
  return a + b;
}

function subtract(a, b) {
  return a - b;
}

try {
  assert.strictEqual(add(2, 3), 5);
  assert.strictEqual(subtract(5, 2), 3);
  assert.notStrictEqual(add(2, 3), 6);
  assert.deepStrictEqual({ a: 1 }, { a: 1 });
  assert.throws(() => {
    throw new Error("错误消息");
  }, Error);
  assert.doesNotThrow(() => {
    /_ 没有错误抛出 _/;
  });

  console.log("所有测试通过");
} catch (error) {
  console.error("测试失败:", error.message);
}
```

使用 assert 模块可以帮助你验证代码的正确性，在开发和调试过程中非常有用。
