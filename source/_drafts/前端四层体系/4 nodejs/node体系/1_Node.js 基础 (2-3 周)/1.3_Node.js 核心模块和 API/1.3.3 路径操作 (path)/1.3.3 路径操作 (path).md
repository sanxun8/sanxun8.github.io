---
title: 1.3.3 路径操作 (path)
---

Node.js 的 path 模块提供了多种工具来处理和转换文件路径。以下是一些常用的路径构建和解析方法：

## 路径构建

- path.join([...paths])：将多个路径片段合并成一个路径，并自动处理不同操作系统的路径分隔符。

```javascript
const path = require("path");

const fullPath = path.join("/users", "john", "documents", "file.txt");
console.log(fullPath); // 输出: /users/john/documents/file.txt
```

- path.resolve([...paths])：将路径或路径片段解析为绝对路径。给定路径序列从右到左解析，直到构建出一个绝对路径。如果没有传入绝对路径，则会使用当前工作目录。

```javascript
const path = require("path");

const absolutePath = path.resolve("users", "john", "documents", "file.txt");
console.log(absolutePath); // 输出: /当前工作目录/users/john/documents/file.txt
```

## 路径解析

- path.basename(path[, ext])：返回路径的最后一部分，可以选择忽略扩展名。

```javascript
const path = require("path");

const baseName = path.basename("/users/john/documents/file.txt");
console.log(baseName); // 输出: file.txt

const baseNameWithoutExt = path.basename(
  "/users/john/documents/file.txt",
  ".txt"
);
console.log(baseNameWithoutExt); // 输出: file
```

- path.dirname(path)：返回路径的目录部分。

```javascript
const path = require("path");

const dirName = path.dirname("/users/john/documents/file.txt");
console.log(dirName); // 输出: /users/john/documents
```

- path.extname(path)：返回路径的扩展名部分。

```javascript
const path = require("path");

const extName = path.extname("/users/john/documents/file.txt");
console.log(extName); // 输出: .txt
```

- path.parse(path)：返回路径的各个部分组成的对象，包括 root、dir、base、ext 和 name。

```javascript
const path = require("path");

const parsedPath = path.parse("/users/john/documents/file.txt");
console.log(parsedPath);

/* 输出:
{
  root: '/',
  dir: '/users/john/documents',
  base: 'file.txt',
  ext: '.txt',
  name: 'file'
}
*/
```

- path.format(pathObject)：将 path.parse 返回的路径对象重新组合成路径字符串。

```javascript
const path = require("path");

const formattedPath = path.format({
  root: "/",
  dir: "/users/john/documents",
  base: "file.txt",
});
console.log(formattedPath); // 输出: /users/john/documents/file.txt
```

## 其他实用方法

- path.isAbsolute(path)：判断给定路径是否是绝对路径。

```javascript
const path = require("path");

console.log(path.isAbsolute("/users/john")); // 输出: true
console.log(path.isAbsolute("documents/file.txt")); // 输出: false
```

- path.relative(from, to)：返回从 from 到 to 的相对路径。

```javascript
const path = require("path");

const relativePath = path.relative(
  "/users/john",
  "/users/john/documents/file.txt"
);
console.log(relativePath); // 输出: documents/file.txt
```

- path.normalize(path)：规范化路径，解决 .. 和 . 片段。

```javascript
const path = require("path");

const normalizedPath = path.normalize(
  "/users/john/../jane/./documents/file.txt"
);
console.log(normalizedPath); // 输出: /users/jane/documents/file.txt
```
