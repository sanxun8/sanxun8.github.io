---
title: 变量的声明和赋值
---

在 JavaScript 中,变量的声明和赋值有以下几种方式:

- 使用 var 关键字声明变量:

```javascript
var x = 10;
var y, z; // 声明多个变量
```

使用 var 声明的变量具有函数作用域,如果不初始化,默认值为 undefined。

- 使用 let 关键字声明变量:

```javascript
let x = 10;
let y, z; // 声明多个变量
```

使用 let 声明的变量具有块级作用域,如果不初始化,默认值为 undefined。

- 使用 const 关键字声明常量:

```javascript
const PI = 3.14159;
const DAYS_IN_WEEK = 7;
```

使用 const 声明的变量是常量,其值不可修改。必须在声明时初始化,且不能重新赋值。

- 不使用关键字声明变量:

```javascript
x = 10; // 隐式声明变量 x
```

这种方式不推荐使用,因为它会创建一个全局变量,通常会导致命名冲突和其他问题。

- 变量重新赋值:

```javascript
let x = 10;
x = 20; // 重新赋值
```

可以对变量进行重新赋值,赋值后变量的值会被更新。

- 多重赋值:

```javascript
let x, y, z;
x = y = z = 10; // 将 10 赋值给 x、y 和 z
```

可以一次性对多个变量赋相同的值。

- 解构赋值:

```javascript
let [a, b, c] = [1, 2, 3];
let { x, y } = { x: 4, y: 5 };
```

使用数组或对象的解构语法快速将值赋给多个变量。

总的来说,JavaScript 提供了多种声明和赋值变量的方式,开发者应根据实际需求选择合适的方式,并注意变量作用域和常量的使用。
