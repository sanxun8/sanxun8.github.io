---
title: 事件的注册和触发
---

在 JavaScript 中,事件是一种非常重要的概念。它们用于处理用户与页面的交互,以及其他各种场景下的状态变化。下面我们来看看如何注册和触发事件:

## 事件注册:

- 使用 addEventListener 方法:

```javascript
const button = document.querySelector("button");

button.addEventListener("click", function () {
  console.log("Button was clicked!");
});
```

- 使用 on 属性:

```javascript
const button = document.querySelector("button");

button.onclick = function () {
  console.log("Button was clicked!");
};
```

## 事件触发:

- 手动触发事件:

```javascript
const button = document.querySelector("button");

// 手动触发 click 事件
button.click();
```

- 使用 dispatchEvent 方法:

```javascript
const button = document.querySelector("button");
const clickEvent = new Event("click");

button.addEventListener("click", function () {
  console.log("Button was clicked!");
});

// 手动触发 click 事件
button.dispatchEvent(clickEvent);
```

## 事件对象:

当事件被触发时,会产生一个事件对象,其中包含了事件的相关信息,如事件类型、目标元素、鼠标坐标等。这个事件对象会作为参数传递给事件处理函数。

```javascript
button.addEventListener("click", function (event) {
  console.log("Button was clicked!");
  console.log(event.type); // 输出: 'click'
  console.log(event.target); // 输出: <button>...</button>
});
```

## 事件传播:

事件在 DOM 树中会经历三个阶段:捕获阶段、目标阶段和冒泡阶段。通过 addEventListener 的第三个参数 useCapture 可以控制事件是在捕获阶段还是冒泡阶段触发。

```javascript
// 在捕获阶段触发事件处理函数
button.addEventListener(
  "click",
  function (event) {
    console.log("Capturing phase");
  },
  true
);

// 在冒泡阶段触发事件处理函数
button.addEventListener(
  "click",
  function (event) {
    console.log("Bubbling phase");
  },
  false
);
```

事件是 JavaScript 编程中非常重要的概念。掌握事件的注册、触发和传播可以帮助我们更好地处理用户交互和页面状态变化。
