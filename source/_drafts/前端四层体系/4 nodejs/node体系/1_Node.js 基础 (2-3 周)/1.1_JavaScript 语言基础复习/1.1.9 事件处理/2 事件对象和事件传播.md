---
title: 事件对象和事件传播
---

在 JavaScript 中,事件对象和事件传播是理解和处理事件的两个重要概念。

## 事件对象:

当一个事件被触发时,浏览器会创建一个事件对象,并将其作为参数传递给事件处理函数。这个事件对象包含了有关该事件的各种信息,比如:

- 事件类型: event.type 返回事件的类型,如 'click'、'keydown' 等。
- 事件目标: event.target 返回触发事件的元素。
- 鼠标坐标: event.clientX、event.clientY 返回事件发生时鼠标的屏幕坐标。
- 键盘信息: event.key、event.keyCode 返回键盘按键的信息。
- 阻止默认行为: event.preventDefault() 可以阻止事件的默认行为。
- 停止事件传播: event.stopPropagation() 可以阻止事件继续向上或向下传播。

## 事件传播:

事件在 DOM 树中会经历三个阶段:

- 捕获阶段: 事件从 window 对象开始向目标元素传播。
- 目标阶段: 事件到达并触发目标元素。
- 冒泡阶段: 事件从目标元素向 window 对象传播。

我们可以通过 addEventListener 的第三个参数 useCapture 来控制事件处理函数是在捕获阶段还是冒泡阶段触发。

```javascript
// 在捕获阶段触发事件处理函数
element.addEventListener(
  "click",
  function (event) {
    console.log("Capturing phase");
  },
  true
);

// 在冒泡阶段触发事件处理函数
element.addEventListener(
  "click",
  function (event) {
    console.log("Bubbling phase");
  },
  false
);
```

使用 event.stopPropagation() 可以阻止事件继续向上或向下传播。

```javascript
element.addEventListener("click", function (event) {
  console.log("Clicked on element");
  event.stopPropagation();
});
```

理解事件对象和事件传播是处理用户交互和页面状态变化的关键。合理地使用这两个概念可以帮助我们编写出更加优秀的 JavaScript 代码。
