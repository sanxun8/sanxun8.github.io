---
title: 字符串与正则表达式
---

在 JavaScript 中,字符串与正则表达式密切相关,正则表达式可以用来匹配、搜索和操作字符串。

## 正则表达式的创建:

正则表达式可以通过两种方式创建:

- 使用字面量语法:

```javascript
const regex = /pattern/flags;
```

- 使用 RegExp 构造函数:

```javascript
const regex = new RegExp("pattern", "flags");
```

## 正则表达式的方法:

- test():

```javascript
const regex = /\d+/;
console.log(regex.test("123")); // 输出: true
console.log(regex.test("abc")); // 输出: false
```

- exec():

```javascript
const regex = /\d+/;
const result = regex.exec("There are 42 apples.");
console.log(result); // 输出: ['42', index: 14, input: 'There are 42 apples.', groups: undefined]
```

## 字符串与正则表达式的方法:

- match():

```javascript
const str = "The quick brown fox jumps over the lazy dog.";
const matches = str.match(/\w+/g);
console.log(matches); // 输出: ['The', 'quick', 'brown', 'fox', 'jumps', 'over', 'the', 'lazy', 'dog']
```

- replace():

```javascript
const str = "The quick brown fox jumps over the lazy dog.";
const newStr = str.replace(/\b\w+\b/g, (match) => match.toUpperCase());
console.log(newStr); // 输出: 'The Quick Brown Fox Jumps Over The Lazy Dog.'
```

- search():

```javascript
const str = "The quick brown fox jumps over the lazy dog.";
const index = str.search(/fox/);
console.log(index); // 输出: 16
```

- split():

```javascript
const str = "apple,banana,cherry";
const fruits = str.split(/,/);
console.log(fruits); // 输出: ['apple', 'banana', 'cherry']
```

正则表达式是一个强大的工具,可以用来处理复杂的字符串操作,如验证输入、提取数据、替换文本等。掌握正则表达式的基本语法和使用方法非常重要。
