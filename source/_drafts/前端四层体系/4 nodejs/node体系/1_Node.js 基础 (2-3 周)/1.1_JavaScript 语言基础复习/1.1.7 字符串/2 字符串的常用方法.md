---
title: 字符串的常用方法
---

除了基本的创建和访问,JavaScript 的字符串还提供了许多常用的方法。下面是一些常用的字符串方法:

- toUpperCase() 和 toLowerCase():

```javascript
const message = "Hello, World!";
console.log(message.toUpperCase()); // 输出: 'HELLO, WORLD!'
console.log(message.toLowerCase()); // 输出: 'hello, world!'
```

- trim(), trimStart() 和 trimEnd():

```javascript
const str = " Hello, World! ";
console.log(str.trim()); // 输出: 'Hello, World!'
console.log(str.trimStart()); // 输出: 'Hello, World! '
console.log(str.trimEnd()); // 输出: ' Hello, World!'
```

- indexOf() 和 lastIndexOf():

```javascript
const str = "The quick brown fox jumps over the lazy dog.";
console.log(str.indexOf("quick")); // 输出: 4
console.log(str.lastIndexOf("the")); // 输出: 31
```

- includes(), startsWith() 和 endsWith():

```javascript
const str = "JavaScript is awesome!";
console.log(str.includes("awesome")); // 输出: true
console.log(str.startsWith("Java")); // 输出: true
console.log(str.endsWith("!")); // 输出: true
```

- padStart() 和 padEnd():

```javascript
const number = "42";
console.log(number.padStart(4, "0")); // 输出: '0042'
console.log(number.padEnd(4, "0")); // 输出: '4200'
```

- repeat():

```javascript
const str = "Ha";
console.log(str.repeat(3)); // 输出: 'HaHaHa'
```

- slice() 和 substring():

```javascript
const str = "The quick brown fox.";
console.log(str.slice(4, 9)); // 输出: 'quick'
console.log(str.substring(4, 9)); // 输出: 'quick'
```

- match() 和 search():

```javascript
const str = "The quick brown fox jumps over the lazy dog.";
console.log(str.match(/quick/)); // 输出: ['quick']
console.log(str.search(/dog/)); // 输出: 37
```

这些只是字符串的一些常用方法,JavaScript 的字符串还有许多其他有用的方法,可以帮助我们更好地处理字符串数据。
