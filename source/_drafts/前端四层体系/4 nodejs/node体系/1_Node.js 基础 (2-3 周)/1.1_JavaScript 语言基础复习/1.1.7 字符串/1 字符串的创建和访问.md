---
title: 字符串的创建和访问
---

在 JavaScript 中,字符串可以通过多种方式创建和访问。

## 字符串创建:

- 使用字符串字面量:

```javascript
const greeting = "Hello, world!";
```

- 使用 String() 构造函数:

```javascript
const message = new String("This is a message.");
```

## 字符串访问:

- 使用索引访问:

```javascript
const str = "JavaScript";
console.log(str[0]); // 输出: 'J'
console.log(str[5]); // 输出: 'S'
```

- 使用 charAt() 方法:

```javascript
const str = "JavaScript";
console.log(str.charAt(0)); // 输出: 'J'
console.log(str.charAt(5)); // 输出: 'S'
```

## 字符串属性和方法:

- length 属性:

```javascript
const message = "Hello, world!";
console.log(message.length); // 输出: 13
```

- concat() 方法:

```javascript
const greeting = "Hello, ";
const name = "Alice";
const fullGreeting = greeting.concat(name);
console.log(fullGreeting); // 输出: 'Hello, Alice'
```

- substring() 方法:

```javascript
const str = "JavaScript is awesome!";
console.log(str.substring(0, 10)); // 输出: 'JavaScript'
console.log(str.substring(11)); // 输出: 'is awesome!'
```

- split() 方法:

```javascript
const str = "apple,banana,cherry";
const fruits = str.split(",");
console.log(fruits); // 输出: ['apple', 'banana', 'cherry']
```

- replace() 方法:

```javascript
const message = "I love JavaScript!";
const updatedMessage = message.replace("JavaScript", "Python");
console.log(updatedMessage); // 输出: 'I love Python!'
```

这些只是字符串的一些基本操作,JavaScript 的字符串还有许多其他有用的方法,可以帮助我们更好地处理字符串数据。
