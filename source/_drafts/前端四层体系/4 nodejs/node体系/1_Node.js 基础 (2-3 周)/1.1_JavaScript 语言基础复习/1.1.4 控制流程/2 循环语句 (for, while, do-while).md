---
title: 循环语句 (for, while, do-while)
---

在 JavaScript 中,我们可以使用以下三种循环语句来重复执行一段代码:

## for 循环:

```javascript
for (初始化表达式; 条件表达式; 更新表达式) {
  // 循环体,重复执行的代码
}
```

- 初始化表达式在循环开始前执行,通常用于初始化循环计数器。
- 条件表达式在每次循环开始时求值,如果结果为 truthy,则继续执行循环体。
- 更新表达式在每次循环结束时执行,通常用于更新循环计数器。

## while 循环:

```javascript
while (条件表达式) {
  // 循环体,重复执行的代码
}
```

- 条件表达式在每次循环开始时求值,如果结果为 truthy,则继续执行循环体。
- 与 for 循环不同,while 循环没有特定的初始化和更新表达式,需要在循环体内手动维护循环变量。

## do-while 循环:

```javascript
do {
  // 循环体,重复执行的代码
} while (条件表达式);
```

- do-while 循环与 while 循环类似,但是它会先执行一次循环体,然后再检查条件表达式。
- 即使条件表达式一开始就为 falsy,do-while 循环也会至少执行一次循环体。

这三种循环语句都可以用于重复执行代码,但它们在使用场景和语法上有所不同。在实际编程中,根据具体需求选择合适的循环语句非常重要。
