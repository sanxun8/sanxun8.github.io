---
title: 对象的继承和原型链
---

JavaScript 中的对象继承是通过原型链实现的。每个对象都有一个指向其原型对象的内部属性 [[Prototype]]。原型对象也有自己的原型,依此类推形成原型链。

## 对象创建和原型链:

- 使用对象字面量创建对象:

```javascript
const person = {
  name: "John Doe",
  greet() {
    console.log(`Hello, my name is ${this.name}`);
  },
};

person.greet(); // 输出: 'Hello, my name is John Doe'
```

在这种情况下,person 对象的原型是 Object.prototype。

- 使用构造函数创建对象:

```javascript
function Person(name) {
  this.name = name;
}

Person.prototype.greet = function () {
  console.log(`Hello, my name is ${this.name}`);
};

const john = new Person("John Doe");
john.greet(); // 输出: 'Hello, my name is John Doe'
```

在这种情况下,john 对象的原型是 Person.prototype。

## 继承:

- 原型链继承:

```javascript
function Animal(name) {
  this.name = name;
}

Animal.prototype.speak = function () {
  console.log(`${this.name} makes a sound.`);
};

function Dog(name) {
  Animal.call(this, name);
}

Dog.prototype = Object.create(Animal.prototype);
Dog.prototype.constructor = Dog;

Dog.prototype.bark = function () {
  console.log(`${this.name} barks.`);
};

const dog = new Dog("Buddy");
dog.speak(); // 输出: 'Buddy makes a sound.'
dog.bark(); // 输出: 'Buddy barks.'
```

- 类继承:

```javascript
class Animal {
  constructor(name) {
    this.name = name;
  }

  speak() {
    console.log(`${this.name} makes a sound.`);
  }
}

class Dog extends Animal {
  constructor(name) {
    super(name);
  }

  bark() {
    console.log(`${this.name} barks.`);
  }
}

const dog = new Dog("Buddy");
dog.speak(); // 输出: 'Buddy makes a sound.'
dog.bark(); // 输出: 'Buddy barks.'
```

理解对象的继承和原型链是 JavaScript 的核心概念之一。继承可以帮助我们重用代码,创建更加结构化和可维护的应用程序。
