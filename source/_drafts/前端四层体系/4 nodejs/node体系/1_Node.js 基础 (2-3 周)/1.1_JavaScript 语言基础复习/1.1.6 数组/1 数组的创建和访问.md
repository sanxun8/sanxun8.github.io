---
title: 数组的创建和访问
---

在 JavaScript 中,数组是一种特殊的数据结构,可以存储多个值。

## 数组的创建:

- 使用数组字面量语法:

```javascript
const myArray = [1, 2, 3, 4, 5];
```

- 使用 Array() 构造函数:

```javascript
const myArray = new Array(1, 2, 3, 4, 5);
```

## 数组的访问:

- 通过索引访问数组元素:

```javascript
const myArray = [10, 20, 30, 40, 50];
console.log(myArray[0]); // 输出: 10
console.log(myArray[2]); // 输出: 30
```

- 数组索引从 0 开始。
- 可以使用负索引来从数组末尾访问元素,例如 myArray[-1] 访问最后一个元素。

**使用 length 属性获取数组长度:**

```javascript
const myArray = [10, 20, 30, 40, 50];
console.log(myArray.length); // 输出: 5
```

## 数组的常用方法:

- push()和 pop()方法:

```javascript
const myArray = [1, 2, 3];
myArray.push(4, 5); // 添加元素到数组末尾, myArray 现在是 [1, 2, 3, 4, 5]
const lastElement = myArray.pop(); // 删除并返回数组最后一个元素, lastElement 是 5, myArray 现在是 [1, 2, 3, 4]
```

- shift()和 unshift()方法:

```javascript
const myArray = [2, 3, 4];
myArray.unshift(1); // 在数组开头添加元素, myArray 现在是 [1, 2, 3, 4]
const firstElement = myArray.shift(); // 删除并返回数组第一个元素, firstElement 是 1, myArray 现在是 [2, 3, 4]
```

- concat()方法:

```javascript
const array1 = [1, 2];
const array2 = [3, 4];
const combinedArray = array1.concat(array2); // combinedArray 是 [1, 2, 3, 4]
```

这些只是数组的一些基本操作,JavaScript 的数组还有许多其他有用的方法,如 map()、filter()、reduce() 等,可以帮助我们更好地操作和处理数组数据。
