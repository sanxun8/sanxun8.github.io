title: 函数作用域和闭包
---

## 函数作用域:

- JavaScript 中有全局作用域和局部作用域。
- 函数内部定义的变量属于该函数的局部作用域,只能在函数内部访问。
- 函数外部定义的变量属于全局作用域,可以在任何地方访问。
- 函数内部可以访问全局作用域中的变量,但全局作用域无法访问函数内部的局部变量。
- 示例:

```javascript
let globalVariable = "I am global";

function myFunction() {
  let localVariable = "I am local";
  console.log(globalVariable); // 可以访问全局变量
  console.log(localVariable); // 可以访问局部变量
}

myFunction();
console.log(globalVariable); // 可以访问全局变量
console.log(localVariable); // 报错,无法访问局部变量
```

## 闭包:

- 闭包是 JavaScript 中一个非常重要的概念。
- 闭包是指一个函数能够访问并记住其外部函数作用域中的变量,即使外部函数已经执行完毕。
- 闭包可以帮助我们实现数据私有化和创建可以记住状态的函数。
- 示例:

```javascript
function outerFunction() {
  let count = 0;

  function innerFunction() {
    count++;
    console.log("Count is:", count);
  }

  return innerFunction;
}

const myFunction = outerFunction();
myFunction(); // 输出: "Count is: 1"
myFunction(); // 输出: "Count is: 2"
myFunction(); // 输出: "Count is: 3"
```

在上面的例子中,innerFunction 是一个闭包,它可以访问并记住 outerFunction 中的 count 变量,即使 outerFunction 已经执行完毕。

闭包是 JavaScript 中非常强大的特性,可以帮助我们编写更加优雅和有效的代码。理解函数作用域和闭包对于掌握 JavaScript 编程非常重要。
