---
title: 函数参数和返回值
---

在 JavaScript 中,函数可以接受参数并返回值。

## 函数参数:

- 函数参数可以是 0 个或多个,用逗号分隔。
- 参数可以是任何 JavaScript 数据类型,包括基本类型(数字、字符串、布尔值等)和引用类型(对象、数组、函数等)。
- 当调用函数时,实参的值会被传递给形参。
- 示例:

```javascript
function greet(name, age) {
  console.log(`Hello, my name is ${name} and I am ${age} years old.`);
}

greet("Alice", 25); // 输出: "Hello, my name is Alice and I am 25 years old."
```

## 函数返回值:

- 函数可以使用 return 语句返回一个值。
- 如果函数没有显式使用 return 语句,则默认返回 undefined。
- 可以返回任何 JavaScript 数据类型,包括基本类型和引用类型。
- 示例:

```javascript
function calculateArea(width, height) {
const area = width \* height;
return area;
}

const result = calculateArea(5, 10);
console.log(result); // 输出: 50
```

## 默认参数和可变长参数:

- 可以为函数参数设置默认值,当调用函数时未传递该参数时,将使用默认值。
- 可以使用 ... 运算符定义可变长参数函数,该函数可以接受任意数量的参数。
- 示例:

```javascript
function greet(name = "Guest", age = 30) {
  console.log(`Hello, my name is ${name} and I am ${age} years old.`);
}

greet(); // 输出: "Hello, my name is Guest and I am 30 years old."
greet("Alice"); // 输出: "Hello, my name is Alice and I am 30 years old."
greet("Bob", 40); // 输出: "Hello, my name is Bob and I am 40 years old."

function sumNumbers(...numbers) {
  let sum = 0;
  for (const num of numbers) {
    sum += num;
  }
  return sum;
}

console.log(sumNumbers(1, 2, 3, 4, 5)); // 输出: 15
```

函数参数和返回值是 JavaScript 编程中非常重要的概念,合理地使用它们可以大大提高代码的可读性和可维护性。
