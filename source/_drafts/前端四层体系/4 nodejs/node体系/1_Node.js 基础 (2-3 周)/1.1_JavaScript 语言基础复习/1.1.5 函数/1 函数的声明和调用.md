---
title: 函数的声明和调用
---

## 在 JavaScript 中,我们可以使用以下两种方式声明函数:

## 函数声明:

```javascript
function functionName(parameter1, parameter2, ...) {
// 函数体,包含要执行的代码
return 返回值;
}
```

- 使用 function 关键字定义函数。
- 函数名称遵循标识符命名规则。
- 函数可以接受 0 个或多个参数,参数之间用逗号分隔。
- 函数体包含要执行的代码,可以使用 return 语句返回一个值。

## 函数表达式:

```javascript
const functionName = function(parameter1, parameter2, ...) {
// 函数体,包含要执行的代码
return 返回值;
};
```

- 将一个匿名函数赋值给一个变量。
- 可以使用变量名来调用这个函数。
- 函数表达式可以是命名函数或匿名函数。

## 调用函数的方式:

```javascript
functionName(argument1, argument2, ...);
```

- 使用函数名(或变量名)后跟一对圆括号来调用函数。
- 括号内传入实参,对应函数定义时的形参。
- 函数执行完成后,如果有 return 语句,则返回相应的值。

## 示例:

```javascript
// 函数声明
function add(a, b) {
  return a + b;
}

// 函数表达式
const multiply = function (a, b) {
  return a * b;
};

console.log(add(3, 4)); // 输出 7
console.log(multiply(5, 6)); // 输出 30
```

在实际编程中,根据需求选择合适的函数声明方式非常重要。函数是 JavaScript 中重要的编程结构,可以帮助我们封装和重用代码,提高程序的可读性和可维护性。
