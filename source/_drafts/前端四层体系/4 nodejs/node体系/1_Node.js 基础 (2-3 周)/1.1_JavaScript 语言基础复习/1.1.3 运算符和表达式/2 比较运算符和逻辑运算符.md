---
title: 比较运算符和逻辑运算符
---

在 JavaScript 中,我们可以使用以下比较运算符和逻辑运算符:

## 比较运算符:

- 等于 (==): 比较两个值是否相等(会进行隐式类型转换)。

```javascript
console.log(5 == 5); // 输出 true
console.log(5 == "5"); // 输出 true
```

- 严格等于 (===): 比较两个值是否相等,且类型必须相同(不会进行隐式类型转换)。

```javascript
console.log(5 === 5); // 输出 true
console.log(5 === "5"); // 输出 false
```

- 不等于 (!=): 比较两个值是否不相等(会进行隐式类型转换)。

```javascript
console.log(5 != 3); // 输出 true
console.log(5 != "5"); // 输出 false
```

- 严格不等于 (!==): 比较两个值是否不相等,且类型必须不同(不会进行隐式类型转换)。

```javascript
console.log(5 !== 3); // 输出 true
console.log(5 !== "5"); // 输出 true
```

- 大于 (>): 比较左侧值是否大于右侧值。

```javascript
console.log(7 > 5); // 输出 true
```

- 小于 (<): 比较左侧值是否小于右侧值。

```javascript
console.log(3 < 7); // 输出 true
```

- 大于等于 (>=): 比较左侧值是否大于或等于右侧值。

```javascript
console.log(8 >= 8); // 输出 true
```

- 小于等于 (<=): 比较左侧值是否小于或等于右侧值。

```javascript
console.log(5 <= 10); // 输出 true
```

## 逻辑运算符:

- 与 (&&): 当左右两个操作数都为 truthy 时,返回 true,否则返回 false。

```javascript
console.log(true && true); // 输出 true
console.log(true && false); // 输出 false
```

- 或 (||): 当左右两个操作数有一个为 truthy 时,返回 true,否则返回 false。

```javascript
console.log(true || false); // 输出 true
console.log(false || false); // 输出 false
```

- 非 (!): 对操作数进行取反操作,true 变为 false,false 变为 true。

```javascript
console.log(!true); // 输出 false
console.log(!false); // 输出 true
```

这些比较运算符和逻辑运算符可以帮助你构建复杂的条件语句和表达式,以实现更复杂的程序逻辑。
