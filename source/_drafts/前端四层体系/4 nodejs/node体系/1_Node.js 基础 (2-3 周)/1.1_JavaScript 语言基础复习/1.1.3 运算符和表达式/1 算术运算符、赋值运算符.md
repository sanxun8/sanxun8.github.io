---
title: 算术运算符、赋值运算符
---

在 JavaScript 中,我们可以使用以下算术运算符和赋值运算符:

## 算术运算符:

- 加法 (+): 用于数字加法和字符串拼接。

```javascript
let x = 5 + 3; // 结果为 8
let y = "Hello" + " " + "world"; // 结果为 "Hello world"
```

- 减法 (-): 用于数字减法。

```javascript
let x = 10 - 4; // 结果为 6
```

- 乘法 (\*): 用于数字乘法。

```javascript
let x = 3 \* 7; // 结果为 21
```

- 除法 (/): 用于数字除法。

```javascript
let x = 15 / 3; // 结果为 5
```

- 取模 (%): 用于获取除法的余数。

```javascript
let x = 17 % 5; // 结果为 2
```

- 自增 (++): 将变量值增加 1。

```javascript
let x = 5;
x++; // 结果为 6
```

- 自减 (--): 将变量值减少 1。

```javascript
let x = 5;
x--; // 结果为 4
```

## 赋值运算符:

- 简单赋值 (=): 将右侧值赋给左侧变量。

```javascript
let x = 10;
```

- 复合赋值: 将运算结果赋给左侧变量。

- 加法赋值 (+=)
- 减法赋值 (-=)
- 乘法赋值 (\*=)
- 除法赋值 (/=)
- 取模赋值 (%=)

```javascript
let x = 5;
x += 3; // 等同于 x = x + 3，结果为 8
x -= 2; // 等同于 x = x - 2，结果为 6
```

这些运算符可以组合使用,构建复杂的表达式。在使用时,请注意运算符的优先级和结合性,以确保得到预期的结果。
