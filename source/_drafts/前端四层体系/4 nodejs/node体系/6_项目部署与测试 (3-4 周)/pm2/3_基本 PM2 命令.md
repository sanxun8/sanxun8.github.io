---
title: 3_基本 PM2 命令
---

好的,非常乐意为您介绍一下 PM2 的基本命令。PM2 是一个非常流行的进程管理器,广泛应用于 Node.js 环境中。下面是一些常用的 PM2 命令:

## 启动应用程序:

```shell
pm2 start app.js
```

这将启动名为 app.js 的应用程序并管理其生命周期。

## 列出所有运行中的应用程序:

```shell
pm2 list
```

该命令将显示所有由 PM2 管理的应用程序的状态和信息。

## 监控应用程序:

```shell
pm2 monit
```

这将启动 PM2 的监控界面,您可以实时查看应用程序的 CPU、内存、日志等信息。

## 停止应用程序:

```shell
pm2 stop app
```

这将停止名为 app 的应用程序。

## 重启应用程序:

```shell
pm2 restart app
```

这将重启名为 app 的应用程序。

## 删除应用程序:

```shell
pm2 delete app
```

这将从 PM2 中删除名为 app 的应用程序。

## 查看日志:

```shell
pm2 logs app
```

这将显示名为 app 的应用程序的日志。

## 设置自动启动:

```
pm2 startup
```

这将为当前用户设置 PM2 在系统启动时自动启动。

## 保存当前配置:

```
pm2 save
```

这将保存当前的 PM2 进程列表,以便在重启后自动恢复。

以上是一些 PM2 的基本命令,您可以根据实际需求进行使用和探索。PM2 还提供了许多高级功能,如负载均衡、监控报警、部署脚本等,可以根据您的具体需求进行深入学习和应用。
